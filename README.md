[![Build AMD64](https://github.com/hep-forge/nlojetxx-feedstock/actions/workflows/autoupload.amd64.yml/badge.svg)](https://github.com/hep-forge/nlojetxx-feedstock/actions/workflows/autoupload.amd64.yml)
[![Build ARM64](https://github.com/hep-forge/nlojetxx-feedstock/actions/workflows/autoupload.arm64.yml/badge.svg)](https://github.com/hep-forge/nlojetxx-feedstock/actions/workflows/autoupload.arm64.yml)

Feedstock for [nlojetxx](https://www.desy.de/~znagy/Site/NLOJet++.html) — part of [hep-forge](https://anaconda.org/hep-forge).

NLOJET++ is a C++ program for calculating LO and NLO order cross section.

## Install

```bash
conda install -c hep-forge -c conda-forge nlojetxx
```

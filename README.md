[![Anaconda Platforms](https://anaconda.org/hep-forge/nlojetxx/badges/platforms.svg)](https://anaconda.org/hep-forge/nlojetxx)

Feedstock for [nlojetxx](https://www.desy.de/~znagy/Site/NLOJet++.html) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

NLOJET++ is a C++ program for calculating LO and NLO order cross section.

## Install

```bash
conda install -c hep-forge -c conda-forge nlojetxx
```
